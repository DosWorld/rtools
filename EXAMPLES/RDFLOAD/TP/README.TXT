# RDFLOAD Example for Turbo Pascal 6/7.

This example show how to load RDF as external dynamic library
and make a call to exported RDF procedure.

MAIN.PAS - main file
RDFLOAD.PAS - unit for loading RDF
TEST.ASM - contains "hello world"-procedure

So, MAIN.PAS load TEST.RDF with RDFLOAD.PAS help and call procedure print_msg.
