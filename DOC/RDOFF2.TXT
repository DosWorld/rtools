 Record types:

 1 - relocation

 3 - symbol export
 5 - BSS size

 Type of segments:

 1 - code segment
 2 - data segment
 3 - BSS

 Very often, segment numbers is:

 0 - code segment
 1 - data segment
 2 - BSS

 Segment numbers gret then 2 is used as link between import records
 and relocation records.

 Teoreticaly, RDOFF2 can have more then 1 code segment and more then 1 data
 segment, but i never meet this situation.

        Null-segment
        ============

 Terminate RDF file and "Segment type" is 0 (and zero-size).
