 Record types:

 0 - Comment
 1 - Relocation (can refer own segment or to import-symbol via RSEG-field)
 2 - Symbol import
 3 - Symbol export
 5 - BSS size
 6 - Segment relocation

 Type of segments:

 1 - code segment
 2 - data segment
 3 - BSS

 Very often, segment numbers is:

 0 - code segment
 1 - data segment
 2 - BSS

 Segment numbers gret then 2 is used as link between import records
 and relocation records.

 Theoretically, RDOFF2 can have more code-data segments (up to 63 in one file),
 but i NEVER meet this situation.

        Null-segment
        ============

 Terminate RDF file, has "Segment type" is 0 (and zero-size).
